## 安装

建立虚拟环境:
```
pip install virtualenv
virtualenv flask
```

建立环境:
```
flask\Scripts\pip install -r requirements.txt(Windows)
flask/bin/pip install -r requirements.txt(Linux)
```

建议使用国内源
```
pip install -i <url>
```

清华：https://pypi.tuna.tsinghua.edu.cn/simple
